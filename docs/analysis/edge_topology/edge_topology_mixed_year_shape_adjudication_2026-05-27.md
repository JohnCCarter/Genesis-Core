## Observed

### 1. Mixed `2023` does have a real December-centered concentration, but it does not collapse into one clean December pocket

On the exact annual `2023` shared-shape surface:

- top combined month = `2023-12` with `38 / 317`
- top continuation month = `2023-12` with `22 / 144`
- top suppression month = `2023-06` with `19 / 173`

So `2023` is genuinely December-led on:

- combined shared-shape mass
- continuation displacement specifically

But it is **not** December-led on suppression.
And the remembered fixed December anchor set mostly does **not** reappear cleanly on the annual surface:

- only `2023-12-20T03:00:00+00:00` overlaps as a counted shared-shape row
- the other four remembered December anchors are absent from the counted annual family

So the annual `2023` read is real, but only partial.

### 2. The only other mixed full year (`2017`) shares June-led suppression, but not the combined or continuation top

The completed mixed-year comparison shows:

- `2017` top combined month = `July`
- `2023` top combined month = `December`
- `2017` top continuation month = `March`
- `2023` top continuation month = `December`
- `2017` top suppression month = `June`
- `2023` top suppression month = `June`

That means the strongest overlap between the two mixed years is:

- shared June suppression leadership

not:

- a shared December-led combined or continuation structure

### 3. The mixed-year label is therefore compositionally split on the currently retained surface

The existing evidence does **not** support one unified mixed-year month-shape story.
Instead it already separates into:

- one shared suppression-side component
- diverging combined and continuation leadership

So the repo already contains evidence that mixed years can agree on one topology component while disagreeing sharply on others.

## Inferred

### 1. Mixed years should not currently be treated as one single topology class

This is the key adjudication result.

The current retained mixed-year evidence is too split for a single-class read because:

- shared June suppression exists across both mixed full years
- but the combined and continuation leaders diverge immediately

So the honest current read is:

- mixed years are **not** one clean reusable topology class on the current surface

### 2. June-led suppression looks like shared mixed-year background, not the decisive mixed-year identity

June suppression is the shared overlap.
That makes it meaningful.
But precisely because it is shared by both mixed years, it is not the thing that distinguishes `2023`.

So June-led suppression is better read as:

- shared mixed-year background structure

not:

- the decisive mixed-year identity signal

### 3. The strongest `2023`-specific signal is the December-led continuation-plus-combined overlay

What actually distinguishes `2023` versus `2017` on the current retained mixed-year surface is:

- December-led combined mass
- December-led continuation displacement

That does **not** promote a new Edge Topology zone by itself.
But it does sharpen the truthful reading of `2023`:

- `2023` is not just a generic mixed year with June suppression
- it is a mixed year with a real December continuation-heavy overlay

### 4. No new zone promotion is justified from this follow-up alone

This follow-up sharpens interpretation.
It does **not** justify:

- a new `ACTIVE` zone
- promotion of a new `WEAK` zone into stronger authority
- rewriting the canonical Phase 5 / Phase 6 registry in this slice

## Unverified

## Verdict

### Mixed-year adjudication result

- **do not treat mixed years as one unified topology class on the current retained surface**

The strongest current read is:

- mixed years share a June-led suppression background component
- mixed years diverge on combined and continuation leadership
- `2023` carries a real December-led continuation-plus-combined overlay
- that overlay remains observational only and non-authoritative
