### Constraints

- read-only over locked trace/backtest artifacts only
- no strategy logic changes
- no config authority changes
- no runtime execution changes
- no fresh traces or backtests generated by this slice
- omission rules must be explicit and deterministic when required fields are unavailable
- the implementation must not claim or encode authority beyond the packet-defined observational artifact surface

### Determinism contract

- Any shuffle/permutation procedure in this slice must be deterministic.
- Ambient or implicit RNG use is forbidden.
- The public entrypoint must require either:
  - a fixed default seed baked into the script contract, or
  - an explicit `--seed` argument with a deterministic default.
- The initial implementation contract for this slice is:
  - expose `--seed`
  - default it to `20260402`
  - require identical outputs across two runs with the same synthetic input and same seed.
- Any future seed-policy change requires a new packet.
